To use this sample extension, you'll need your own DropBox API access token. The easiest way to do this is by creating your own API app:

  1. Open this link in your browser, and log in using your account
  2. Click on "Create App", then select "Dropbox API app"
  3. Now go on with the configuration, choosing the app permissions and access restrictions to your DropBox folder
  4. Now, click on the "Create App" button.
